# KMFC — Посадочные страницы для контекстной рекламы

## Обзор проекта

**Клиент:** Казахстанская Микрофинансовая Компания (KMFC)
**Сайт:** https://kmfc.kz/ (Тильда)
**Задача:** 4 посадочные страницы для PPC-кампаний (Google Ads / Яндекс.Директ)
**Размещение:** Субдомен (напр. `landing.kmfc.kz`)
**Индексация:** noindex, nofollow
**ГЕО:** Алматы (основной) + Актобе (филиал)
**Целевая аудитория:** Физлица + бизнес (ИП/ТОО)
**Цель:** Максимальная конверсия в заявку

---

## 4 посадочные страницы

| # | Тип залога | URL-slug | Ключевые запросы (примеры) |
|---|-----------|----------|---------------------------|
| 1 | Недвижимость (квартира, дом) | `/nedvizhimost` | кредит под залог квартиры алматы, займ под залог дома |
| 2 | Земельный участок | `/zemelnyj-uchastok` | кредит под залог земли алматы, займ под залог участка |
| 3 | Автомобиль | `/avto` | кредит под залог авто алматы, займ под залог машины |
| 4 | Спецтехника | `/spectehnika` | кредит под залог спецтехники, залог экскаватора |

## Структура каждой посадочной (wireframe)

### Hero-секция
- Заголовок H1 (адаптирован под тип залога)
- Подзаголовок с УТП
- CTA-кнопка "Оставить заявку"
- Ключевые цифры: сумма / ставка / срок / скорость одобрения

### Секция "Условия"
- Сумма кредита (от — до)
- Процентная ставка
- Срок кредитования
- Срок рассмотрения заявки
- Требования к залогу (специфичные для типа)

### Секция "Преимущества"
- 4-6 карточек с иконками
- Примеры: быстрое одобрение, без скрытых комиссий, гибкий график, досрочное погашение

### Секция "Как получить кредит" (шаги)
- Шаг 1: Оставьте заявку
- Шаг 2: Оценка залога
- Шаг 3: Одобрение
- Шаг 4: Получение денег

### Секция "Другие виды залога" (СКВОЗНАЯ)
- 4 карточки с типами залога
- Текущий тип залога визуально выделен (active state)
- Линки на остальные 3 посадочные

### Форма заявки (лид-форма)
- Имя
- Телефон (маска +7)
- Тип залога (предвыбран текущий)
- Сумма (опционально)
- CTA: "Получить консультацию"
- Текст о согласии на обработку персональных данных

### Секция "Контакты / О компании"
- Краткое описание KMFC
- Лицензия / регуляторная информация
- Адреса: Алматы + Актобе
- Телефоны, WhatsApp

### Footer (минимальный)
- Логотип
- Копирайт
- Ссылка на политику конфиденциальности

## Технический стек

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Формы:** React Hook Form + Server Actions или внешний сервис
- **Analytics:** Google Analytics 4 + Яндекс.Метрика
- **Deploy:** Vercel (субдомен `landing.kmfc.kz`)
- **Язык:** Русский (основной), казахский (опционально)

## Конверсионная стратегия

### Точки захвата лида (минимум 3 на страницу)
1. **Hero CTA** — кнопка "Оставить заявку" в первом экране
2. **Sticky CTA** — фиксированная кнопка/баннер на мобильных
3. **Форма** — полноценная форма заявки в нижней части страницы
4. **WhatsApp CTA** — плавающая кнопка WhatsApp (опционально)

### Конверсионные элементы
- Социальное доказательство: "Более X клиентов получили кредит"
- Срочность: "Одобрение за 1 день"
- Простота: "Всего 2 документа"
- Доверие: лицензия, года на рынке
- Телефон кликабельный (tel:)

### Mobile-first приоритеты
- Click-to-call на телефоне
- Click-to-WhatsApp
- Sticky CTA внизу экрана
- Форма без лишних полей (3-4 максимум)
- Быстрая загрузка (< 3 сек)

## Отслеживание конверсий

### Цели для аналитики
1. `form_submit` — отправка формы заявки
2. `phone_click` — клик по телефону
3. `whatsapp_click` — клик по WhatsApp
4. `cta_click` — клик по CTA в hero
5. `cross_section_click` — клик по другому типу залога

### UTM-параметры
Страницы должны корректно сохранять UTM из URL рекламных кампаний и передавать в форму заявки как скрытые поля.
